# online-Food-Ordering-App-UI

### Project Overview

The Online Food Ordering App UI is a mobile application interface designed using Figma. The project focuses on creating a modern, attractive, and user-friendly food ordering experience with easy navigation, food categorization, and responsive mobile screens.

### Features

- Splash Screen
- Login Screen
- Welcome Screen
- Food Category Selection
- Vegetarian Menu
- Non-Vegetarian Menu
- Food Search Functionality
- Popular Food Section
- Personalized Recommendations
- User Profile Access
- Bottom Navigation Bar
- Responsive Mobile Design

### Tools Used

- Figma

### Design Screens

- Splash Screen
- Login Screen
- Welcome Screen
- Category Selection Screen
- Vegetarian Menu Screen
- Non-Vegetarian Menu Screen
- Food Listing Screen
- Home Dashboard

### Objectives

- Create an intuitive food ordering interface.
- Enhance user experience through simple navigation.
- Provide quick access to food categories and menu items.
- Design a visually appealing and responsive mobile application.

### Project Highlights

- Modern UI Design
- User-Friendly Interface
- Consistent Color Scheme
- Mobile-First Design
- Easy Navigation
- Interactive User Experience
